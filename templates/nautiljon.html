Comment utiliser l'enrichissement Nautiljon?

1. Enrichir une Série

Recherchez le titre du manga sur Nautiljon ou sélectionnez une série de votre bibliothèque. L'application récupérera automatiquement:

  • 📊 Nombre total de volumes
  • 🇫🇷 Nombre de tomes français
  • 📕 Éditeur français
  • 👨 Mangaka/Auteur
  • 🔄 Statut (En cours, Terminé, Pausé)
  • 📅 Années de publication

2. Enrichissement par Lot

Sélectionnez plusieurs séries et enrichissez-les toutes en une seule opération. Parfait pour initialiser votre collection!

3. Enrichir une Bibliothèque Complète

Enrichissez toutes les séries d'une bibliothèque en une seule opération. Si vous le souhaitez, l'application scanera d'abord la bibliothèque pour détecter les nouvelles séries.

Conseils

  • 💡 L'enrichissement est rapide et sans risque - les données sont stockées localement
  • 💾 Les données sont sauvegardées dans votre base de données
  • 🔄 Vous pouvez enrichir à nouveau pour mettre à jour les infos
  • ⚠️ Certains mangas très récents ou obscurs peuvent ne pas être trouvés
